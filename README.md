# Ирка-всезнайка

Персональная академия Ирины по устройству бытовой техники и безопасному подбору запчастей ZIP161.

## Что включено

- 11 учебных маршрутов и более 70 разобранных категорий деталей;
- устройство приборов от общего принципа к системам;
- примеры и прямые ссылки на ZIP161;
- рабочий справочник и шестишаговый чек-лист подбора;
- более 50 карточек с интервальным повторением;
- мини-кейсы продавца;
- персональная ИИ-наставница Миса (Gemini API);
- локальное хранение прогресса без регистрации;
- исходная база материалов в папке `materials/source-base`.

## Запуск на этом компьютере

Откройте **«Запустить Ирку-всезнайку.cmd»**. При первом запуске программа попросит существующий Gemini API key и сохранит его только в локальном файле `.env.local`. Этот файл исключён из Git.

Также можно открыть `index.html` напрямую в браузере — все уроки, справочник и повторение работают без сервера. Чат потребует настройки ключа Gemini через интерфейс.

## Размещение в интернете

### GitHub Pages (рекомендуется)

Вся функциональность, включая чат, работает без сервера — Gemini API вызывается напрямую из браузера:

1. Создайте GitHub-репозиторий.
2. Загрузите содержимое папки `irka-vseznaika`.
3. В настройках репозитория включите GitHub Pages (вкладка Pages → Source: main branch).
4. Готово! Чат с Мисой запросит ключ Gemini при первом использовании.

### Vercel

1. Загрузите содержимое папки в закрытый GitHub-репозиторий.
2. Импортируйте репозиторий в Vercel.
3. В Environment Variables добавьте `GEMINI_API_KEY` и `GEMINI_MODEL=gemini-2.5-flash-lite`.
4. Выполните Deploy.

### Локальный сервер

```bash
node server.js
```

Сервер нужен только для старого варианта с серверным прокси Gemini. Клиентский вариант (рекомендуемый) работает без него.

## Получение ключа Gemini

1. Откройте [Google AI Studio](https://aistudio.google.com/apikey).
2. Войдите под Google-аккаунтом.
3. Нажмите «Create API key».
4. Скопируйте ключ и вставьте в приложение при запросе.

Ключ хранится только в `localStorage` браузера и никуда не отправляется, кроме API Google.

## Важное ограничение

Материалы обучают продавца понимать системы и собирать данные для подбора. Они не заменяют диагностику мастера и не являются инструкцией по самостоятельным работам с газом, сетевым напряжением, давлением или герметичным холодильным контуром.
